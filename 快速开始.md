# 🚀 快速开始指南

## 第一次使用？按照以下步骤操作：

### 步骤 1：启动服务

在终端中运行：

```bash
./start.sh
```

这个脚本会自动：
- ✅ 检查并创建虚拟环境（如果不存在）
- ✅ 安装所有依赖包
- ✅ 启动 API 服务

### 步骤 2：访问 API 文档

服务启动后，在浏览器中打开：

**http://localhost:8000/docs**

这是一个交互式的 API 文档页面，你可以：
- 查看所有 API 端点
- 直接在页面上测试 API
- 查看请求和响应格式

### 步骤 3：测试 API

#### 方法 1：在浏览器中测试
访问 http://localhost:8000/docs，点击 `/constitution/estimate` 端点，点击 "Try it out"，输入测试文本，点击 "Execute"。

#### 方法 2：使用 curl 命令

打开**新的终端窗口**（保持服务在第一个终端运行），运行：

```bash
# 测试健康检查
curl http://localhost:8000/health

# 测试体质判定（气虚质示例）
curl -X POST "http://localhost:8000/constitution/estimate" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "我最近总是感觉很累，容易疲劳，说话声音也比较低，稍微动一下就出汗，还经常感冒。",
    "meta": {
      "age": 35,
      "sex": "M"
    }
  }'
```

## 常见问题

### Q: 如何停止服务？
A: 在运行服务的终端窗口中，按 `Ctrl + C`

### Q: 如何重新启动服务？
A: 再次运行 `./start.sh` 即可

### Q: 端口 8000 被占用怎么办？
A: 编辑 `main.py` 文件，修改最后一行的 `port=8000` 为其他端口号（如 8001）

### Q: 虚拟环境是什么？
A: 虚拟环境是一个独立的 Python 环境，可以避免不同项目的依赖包冲突。项目已经自动创建了虚拟环境，你不需要手动管理。
